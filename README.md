# BARBARUS API

Minimal Vercel API for the `THE DARK SIDE` page.

## Endpoint

- `POST /api/dark-side`

## Local env

Create `.env.local` from `.env.example` and set:

```env
OPENAI_API_KEY=sk-...
```

## Deploy

Import this folder as a separate Vercel project and set `OPENAI_API_KEY` in the Vercel project settings.
